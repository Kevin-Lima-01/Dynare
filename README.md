# Dynare Models for Advanced Macroeconomics

This repository contains a collection of **Dynare** codes developed for advanced macroeconomics coursework. Each script specifies and simulates **New Keynesian (NK)** and **Real Business Cycle (RBC)** models, focusing on the dynamic effects of exogenous shocks.

## Key Features

- **Model Frameworks:**
  - New Keynesian (NK) models with nominal rigidities
  - Real Business Cycle (RBC) models

- **Shock Analysis:**
  The primary objective is to specify a shock to a given variable and estimate its macroeconomic impact. The focus is on three types of shocks:
  - 🔹 **Fiscal shocks**
  - 🔹 **Monetary shocks**
  - 🔹 **Technology shocks**

- **Impulse Response Functions (IRFs):**
  All simulations include impulse response analysis to visualize the transmission mechanisms and the persistence of shocks over time.
